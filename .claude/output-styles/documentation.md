Every response must include documentation:
- Code changes → include updated docstrings/comments
- Architecture decisions → include ADR summary
- API changes → include OpenAPI/GraphQL schema updates
- New features → include README section
- Bug fixes → include root cause analysis
Never ship code without shipping its documentation.
